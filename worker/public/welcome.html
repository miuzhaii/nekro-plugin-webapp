Core Features
Architecture Highlights

Async Task Engine

后台独立运行，主对话完全不阻塞。支持进度追踪、中途反馈与任务取消。

Streaming Tool Call

纯文本协议输出，操作块完成后立即执行。错误即时打断，节省 Token 开销。

Progressive Tolerance

DIFF 渐进式容错机制，自动修复空白差异。连续失败时智能降级为全量覆写。

Virtual File System

完整的项目管理能力，支持多文件组织。自动提取导出信息，防止模块引用错误。

Permission Isolation

管理密钥与访问密钥分离，符合最小权限原则。支持多用户共享与协作。

Auto Iteration

编译失败自动重试，最多 20 轮迭代修复。类型检查可选跳过，避免第三方库误报阻塞。
